# Evaluation
Follow the `uccs-gen/evaluate.sh` file to test the two stages separately.
Follow the `uccs-gen/evaluate_pipeline.sh` file to test the pipeline system.

```bash
cd uccs-gen
bash evaluate.sh
bash evaluate_pipeline.sh
```
